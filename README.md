-Algorithm Visualizer là website mô phỏng trực quan các thuật toán bằng HTML, CSS và JavaScript thuần. Dự án được xây dựng nhằm hỗ trợ học tập và trình bày thuật toán thông qua animation sinh động, dễ hiểu và có tính tương tác cao.
Tính năng
  Mô phỏng thuật toán trực quan bằng animation
  Giao diện hiện đại, responsive
  Hỗ trợ Dark Mode / Light Mode
  Điều khiển animation: Chạy, Tạm Dừng, Đặt Lại, Ngẫu Nhiên
  Điều chỉnh tốc độ thuật toán
  Nhập dữ liệu thủ công
  Hiển thị thống kê và giả mã thuật toán
  Animation thời gian thực
Thuật toán hỗ trợ
  Sắp xếp
    Bubble Sort (Sắp Xếp Nổi Bọt)
    Selection Sort (Sắp Xếp Chọn)
  Tìm kiếm
    Binary Search (Tìm Kiếm Nhị Phân)
  Cấu trúc dữ liệu
    Stack (Ngăn Xếp)
    Queue (Hàng Đợi)
  Đồ thị
    BFS (Breadth First Search)
    DFS (Depth First Search)
Công nghệ sử dụng
  HTML5
  CSS3
  JavaScript (Vanilla JS)
